Mes applications

Voila mes plus gros projets auquels je consacre beaucoup de temps

A2L

L'A2L se modernise avec l'arrivée d'une application iOS et un site web !. Les adhérents pourront maintenant virtualiser leurs cartes, et gérer leurs avantages en direct. Les membres du CA de l'A2L auront à leur disposition un ensemble d'outil pour gérer tous les adhérents. Ce projet est open-source !

  • Statut : Site web publié et en fonctionnement
  • Modèle économique : 100% gratuite
  • Accéder au codes sources du projet : Application iOS, Site web, serveur

Error Library

Une appliaction incontournable pour les developpeurs qui leurs permet de gerer très facilement les codes erreurs issus de leurs applications. En une recherche, on y retrouve le nom de l'appli, le nom du fichier concerné, la ligne, une description etc...

  • Statut : En developpement ...
  • Modèle économique : Gratuite avec Pubs. Possibilité de les supprimer et débloquer un espace de stockage illimité ainsi que me soutenir(1.99€)
  • Je veux faire partie des bêtas testeurs à sa sortie (début 2019)

Officiels du Badminton

Centré sur l'arbitrage au badminton, cette application est destiné au arbitre, mais pas que. De nouvelles fonctionnalités tel que le mode 'spectateur' arriveront très vite. Elle se veut d'être complète et en association avec la Fédération Française de badminton

  • Statut : En developpement ...
  • Modèle économique : Gratuites avec Pubs. Mode payant qui permet de supprimer les pubs et me soutenir(2,50€)
  • Je veux faire partie des bêtas testeurs à sa sortie (2019)
